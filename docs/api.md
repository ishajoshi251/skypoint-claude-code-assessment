# TalentBridge — API Reference

Interactive OpenAPI docs are auto-generated by FastAPI and available at:

- **Swagger UI:** http://localhost:8000/docs
- **ReDoc:** http://localhost:8000/redoc
- **OpenAPI JSON:** http://localhost:8000/openapi.json

## Auth Flow Notes

All protected endpoints require `Authorization: Bearer <access_token>` header.

| Endpoint | Method | Auth | Description |
|---|---|---|---|
| `/api/v1/auth/register` | POST | — | Register new user |
| `/api/v1/auth/login` | POST | — | Login, returns access token + sets refresh cookie |
| `/api/v1/auth/refresh` | POST | refresh cookie | Rotate refresh token, return new access token |
| `/api/v1/auth/logout` | POST | Bearer | Revoke refresh token |
| `/api/v1/jobs` | GET/POST | Bearer | List jobs / create (HR only) |
| `/api/v1/applications` | GET/POST | Bearer | List / submit applications |
| `/api/v1/candidates/search` | POST | Bearer (HR) | Smart-search candidates |
| `/api/v1/resumes/upload` | POST | Bearer (Candidate) | Upload + parse resume |
| `/api/v1/matching/score` | GET | Bearer | Get match score for job+candidate |
| `/api/v1/analytics` | GET | Bearer (HR) | HR dashboard analytics |
